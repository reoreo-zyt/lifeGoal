| 稀有度 | 星级 | 基础数值范围 | 某项可突出至 | 统率值 | 战法距离 |

| 白 | ★★ | 40-60 | 70+ | 1.5 | 2 |

| 紫 | ★★★★ | 60-72 | 88 | 2.5 | 3-5 |

- **成长值**：0.98 - 3.00，每位武将各项成长独立设定
- **队伍统率**：1.5 - 3.5，稀有度越高统率越大，用于队伍编成上限
- **战法距离**：主动战法独有，稀有度越高距离越远
- **战法类型**：指挥（战斗开始/每回合触发）、主动（需主动释放）、被动（常驻生效）
- **追击/反击/连击**：特殊机制，可附加于任意战法类型

## 战法类型说明

- **指挥**：战斗开始时或每回合开始时自动触发，覆盖我方/敌方全体或特定目标，常驻全场或持续多回合
- **主动**：需要主动释放，有冷却时间、释放距离和发动概率，战斗中可多次使用
- **被动**：常驻生效，无需主动操作，不占用行动回合

### 特殊机制

- **追击**：普攻后有概率追加一次额外伤害
- **反击**：受到攻击时，有概率回击攻击者
- **连击**：普攻时额外追加一次普攻
- **增伤**：满足特定条件时获得伤害加成
- **攻心**：攻击时削弱敌方属性

|------|------|---------|
| 震慑 | 无法行动 | 被动免疫/持续结束 |
| 怯战 | 无法物理攻击 | 被动免疫/持续结束 |
| 沉默 | 无法发动主动战法 | 被动免疫/持续结束 |
| 混乱 | 攻击己方单位 | 被动免疫/持续结束 |
| 暴走 | 随机攻击目标（包括友军或敌军） | 被动免疫/持续结束 |

| 减伤 | buff | 伤害减免 X% |

## 数值设计原则

1. **突出差异化**：每位武将至少一项属性显著高于同稀有度武将
2. **机制绑定稀有度**：追击/反击/连击等高级机制优先分配给紫将以上
3. **缺陷即特色**：低属性不等于弱，而是为特定玩法服务
4. **羁绊强化组合**：配对武将形成1+1>2的效果
5. **动态平衡**：高速武将用低防/低血换取先手优势
