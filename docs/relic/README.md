# 遗物系统设计

> 本文档定义了游戏中「遗物」的设计规范，包括分类、数值、获取方式及与现有系统的交互。

---

## 概述

遗物是玩家在地图探索中于「宝物」节点获得的可收集物品，属于**被动常驻的全军效果或功能性效果**。与技能/羁绊的「战斗中生效」不同，遗物提供全局增益、资源加成、探索辅助或特殊机制，影响整个跑图流程。

**遗物与装备的区分**（重要）：

| 维度 | 装备（未来） | 遗物 |
|------|------------|------|
| 作用对象 | 单个武将 | 全体武将 |
| 效果类型 | 个体数值强化（攻击/防御/速度等 +X） | 全军增益或功能性 |
| 叠加方式 | 同一武将装备槽位 | 效果叠加，不同名即可并存 |
| 生效时机 | 战斗中（随武将上场） | 全局/跑图中（整个run有效） |
| 核心价值 | 强化特定武将 | 影响整体策略方向和资源节奏 |

> 遗物**不提供**单个武将的数值强化（如"秦琼攻击+12%"）。这类效果留给未来的装备系统实现。

## 遗物数据结构

```typescript
// src/relics/types.ts

export type RelicRarity = "common" | "rare" | "epic" | "legendary";

/**
 * 遗物效果分类
 * - teamBuff:   全军增益（全队攻击/防御/速度等提升）
 * - debuff:     全局给敌方施加的减益
 * - resource:   资源获取、商店相关
 * - survival:   全队生存、续航、恢复
 * - explore:    探索辅助、地图可见性
 * - special:    特殊机制（Boss特攻、胜利加成等）
 */
export type RelicCategory =
  | "teamBuff"
  | "debuff"
  | "resource"
  | "survival"
  | "explore"
  | "special";

export interface RelicEffect {
  /** 效果描述（用于Tooltip） */
  description: string;
  /** 效果影响的目标范围 */
  scope: "team" | "enemy" | "global" | "self"; // self 仅用于触发式效果（濒死回复等）
}

export interface RelicSetBonus {
  pieces: 2 | 4;
  description: string;
}

export interface RelicSet {
  id: string;
  name: string;
  books: string[];       // 对应典籍名称
  description: string;  // 套装主题描述
  pieces: RelicSetBonus[];
}

export interface Relic {
  id: string;
  name: string;
  rarity: RelicRarity;
  setId: string | null;
  category: RelicCategory;
  tags: string[];        // 如 ["attack", "team"]
  icon: string;
  effect: RelicEffect;
  source: "treasure" | "elite" | "boss" | "event";
  lore: string;          // 历史典籍背景描述
}
```

## 稀有度与数值框架

| 稀有度 | 颜色 | 稀有度权重 | 数值幅度 | 获取来源 |
|--------|------|-----------|---------|---------|
| 白（common） | 灰 | 45% | 全队+5-12% | 普通宝物节点 |
| 绿（rare） | 绿 | 30% | 全队+10-18% | 普通/精英宝物 |
| 紫（epic） | 紫 | 18% | 全队+15-30% 或独特机制 | 精英/事件 |
| 金（legendary） | 金 | 7% | 极强效果或独特机制 | Boss/高级事件 |

> 注意：遗物数值幅度**以全队为基准**，等效于单武将装备效果×3（全队3人），保持装备系统的单武将强化优势。

## 遗物套装设计

遗物套装以唐朝及历代正史、典籍为名，组成4套主题系列，每套4件。集齐2件/4件分别触发对应效果。

---

### 套装一：「贞观系列」— 武备与攻战

**典籍来源**：《六韬》《三略》《李卫公问对》《阴陵令》

**背景**：贞观十七年，太宗李世民命图画开国功臣于凌烟阁二十四功臣，功业彪炳者皆以武略传世。此套典籍涵盖中国兵学两千年之精华：太公吕望《六韬》论政治、军事、经济之综合国略；黄石公《三略》专述揽英雄、收人心之要诀；李靖与唐太宗《李卫公问对》以问答形式穷尽奇正攻守之变；《阴陵令》则为陈婴佐刘邦起兵之遗策，述以弱胜强、以少敌众之法。太宗以此四书训诫天策府诸将，以为攻战之本。此套遗物汇聚先秦至唐初兵学之精要，合而言之，正是太宗"以武定天下"之雄图。

| 其三 | 「李卫公问对」 | 紫 | 全队追击概率+18%，追击追加伤害+25% | 精英 |

### 套装二：「开元系列」— 谋略与控制

**典籍来源**：《贞观政要》《帝范》《臣轨》《新唐书》

**背景**：开元年间，唐玄宗以太宗为法，励精图治。姚崇、宋璟相继为相，以贞观之政为镜，创开元盛世。《贞观政要》吴竞所撰，网罗太宗与魏征、房玄龄、杜如晦等人问答之语，记治国以民为本、兼听则明之心法；《帝范》则太宗躬笔，亲述为君之道十二篇，以训诫太子；武则天临朝时命百官习《臣轨》，以忠义约束群臣；至宋仁宗时，欧阳修、宋祁奉诏修《新唐书》，以理学笔法重撰唐代兴亡之鉴。此四书横跨三百年，上承太宗之治，中经女皇之政，下接宋代史笔，以"谋略"二字贯穿始终——谋国、谋君、谋臣、谋史，构成一套完整的政治智慧体系。

| 部件 | 典籍名 | 稀有度 | 全队效果 | 来源 |
|------|--------|--------|---------|------|
| 其一 | 「贞观政要」 | 绿 | 全队谋略+12%，敌方全体策防-8%全场 | 宝物 |
| 其二 | 「帝范」 | 绿 | 全队主动战法概率+12%，速度+6% | 宝物 |
| 其三 | 「臣轨」 | 紫 | 每回合敌方随机2人谋略-10%持续2回合 | 精英 |

| **2件套** | — | — | 全队谋略+18%，敌方全体策防-15%全场，策略伤害+15% | — |

### 套装三：「典谟系列」— 史学与文献

**典籍来源**：《旧唐书》《资治通鉴》《史通》《艺文类聚》

**背景**：史学为唐人立国之学，刘知几所谓"史才三长"——才、学、识，为士人最高追求。后晋刘昫等奉敕修《旧唐书》，以唐代当朝史料为本，记本朝之史于本朝之人，详尽而真切；宋司马光承神宗之命，历十九年成《资治通鉴》，以编年体统合十七史，上起战国下及五代，以史为镜，以鉴来者；刘知几自著《史通》，为中国第一部史学批评专著，批判前人修史之失，立史家独立之精神；欧阳询等人奉敕编《艺文类聚》，汇先秦至隋代诗文典故于一炉，分门别类，以供文士采撷。此四书合为一系，代表中国史学四大门类——本朝正史、历代通鉴、史学理论、文献汇编——合而观之，正是"以史为鉴"这一中华文明核心精神的完整呈现。

| 部件 | 典籍名 | 稀有度 | 全队效果 | 来源 |
|------|--------|--------|---------|------|
| 其一 | 「旧唐书」 | 绿 | 全队防御+12%，每回合全队恢复2%兵力 | 宝物 |
| 其二 | 「资治通鉴」 | 绿 | 战斗开始时驱散全队各1个debuff | 宝物 |
| 其三 | 「史通」 | 紫 | 每场战斗首次我方有人受到致命伤害时，免疫此次伤害并获得25%最大兵力的护盾 | 精英 |
| 其四 | 「艺文类聚」 | 紫 | 敌方全体速度-12%，全队获得8%减伤 | 精英 |
| **2件套** | — | — | 全队物理减伤+15%，策略减伤+15%，每回合恢复2%兵力 | — |
| **4件套** | — | — | 全队免疫控制效果（怯战/震慑/沉默）；战斗结束时全队恢复12%兵力 | — |

### 套装四：「武德系列」— 天下武功

**典籍来源**：《秦王破阵乐》《太极拳谱》《少林武僧记》《裴氏兵法》

**背景**：武德乃唐高祖李渊开国之年号，亦为中华武学之精神底蕴。秦王李世民平定刘武周时，军中乐师制成《破阵乐》，以百二十人披甲执戟而舞，气势磅礴，声震四方，玄奘取经途中将此曲携至印度，玄宗亲注《秦王破阵乐》以教习武人；太极拳谱则源自道家内丹修炼之术，以柔克刚、借力打力，后与少林武学相融，成为中国武学之最高哲理；少林武僧助唐太宗平定王世充之乱后，李世民赐紫袈裟、田产，少林武僧记即记此殊遇与少林武学传承之源流；裴氏兵法出自裴仁基、裴行俨父子，裴氏三代为将，李密、王伯当皆出其门下，以"以奇用兵"著称。此四典籍合为一系，从战阵之舞、内家之法、外家之功、奇兵之略四个维度，构成唐人尚武之完整图景，亦为中华武学千年之根脉所在。

| 部件 | 典籍名 | 稀有度 | 全队效果 | 来源 |
|------|--------|--------|---------|------|
| 其一 | 「秦王破阵乐」 | 绿 | 全队连击概率+15%，攻击+8% | 宝物 |
| 其二 | 「太极拳谱」 | 绿 | 全队速度+15%，先手概率+15% | 宝物 |
| 其三 | 「少林武僧记」 | 紫 | 全队反击概率+25%，反击伤害+40% | 精英 |
| 其四 | 「裴氏兵法」 | 紫 | 全队追击概率+20%；追击追加伤害额外+30%（基于速度差） | 精英 |
| **2件套** | — | — | 全队速度+20%，先手概率+20%，连击概率+15% | — |
| **4件套** | — | — | 全队追击触发后35%概率追加一次连击；击杀目标后全队获得1回合25%增伤 | — |

## 散件遗物设计

散件遗物均为**全军效果**或**功能性效果**，分为以下六类。

### 一、全军增益（teamBuff）

全队属性提升或战斗机制强化。

| 遗物名 | 稀有度 | 全队效果 | 典籍/背景 |
|--------|--------|---------|----------|
| 「虎符」 | 白 | 全队攻击+8% | 调兵信物，太尉与皇帝各执其一。大军将出，师出有名。 |
| 「赤兔马」 | 绿 | 全队速度+15%，先手概率+12% | 吕布、关羽皆曾跨此名驹，日行千里。李元吉珍藏此马，以示荣耀。 |
| 「风林火山」 | 紫 | 全队必定先手；每回合首次行动驱散自身1个debuff | 语出《孙子兵法》，李世民以此八字训玄甲军："其疾如风，其徐如林。" |
| 「天子剑」 | 紫 | 全队对HP>50%的目标伤害+25% | 李世民佩剑，玄武门之变中亲手斩杀建成、元吉。此后此剑藏于秦王府，传为天子之器。 |
| 「裴氏兵法」 | 紫 | 全队追击概率+20%，追击追加伤害额外+30%（基于速度差） | 裴仁基、裴行俨父子所创，以"以快制慢"为核心，后为李靖收入兵法体系。 |
| 「秦王破阵乐」 | 绿 | 全队连击概率+15%，攻击+8% | 李世民破刘武周时，军中奏《破阵乐》，舞者百二十人，被甲执戟，以象战阵之容。 |
| 「太极拳谱」 | 绿 | 全队速度+15%，先手概率+15% | 道家内修功法，讲究以柔克刚、四两拨千斤。少林武僧将此法与搏击之术融合，传于后世。 |

## 获取与叠加规则

### 获取来源

| 节点类型 | 掉落的遗物稀有度范围 | 额外说明 |
|---------|---------------------|---------|
| 普通宝物节点 | 白/绿 | 必有1件 |
| 精英战斗后 | 绿/紫 | 必有1件，紫概率35% |
| Boss战后 | 紫/金 | 必有1件，金概率25% |
| 事件选择 | 白/绿/紫 | 视事件难度 |
| 商店购买 | 白/绿/紫/金 | 可选，消耗金币 |

### 叠加规则

1. **同名遗物不叠加**：已持有的遗物无法再次获得
2. **不同遗物可叠加**：效果独立计算，可叠加
3. **套装按件数触发**：持有同套装部件达到指定数量时，触发套装效果
4. **同类效果可叠加**：全军攻击+X与全军攻击+Y同时存在时，数值相加

## 获取动画规格

遗物获取时弹出 **遗物获得展示（RelicAcquisitionModal）**，按稀有度分级施放不同强度动画。所有动画时长约 2.8 秒，自动关闭或点击确认消失。

### 动画层级总览

| 稀有度 | 动画等级 | 时长 | 特效关键词 |
|--------|---------|------|-----------|
| 白（common） | 简约 | ~1.5s | 淡入 + 微缩放 |
| 绿（uncommon） | 普通 | ~2s | 缩放弹出 + 绿色微光 |
| 紫（rare/epic） | 酷炫 | ~2.8s | 紫光爆发 + 粒子飞散 + 震动 |
| 金（legendary） | 史诗 | ~3.5s | 金光爆发 + 光线辐射 + 屏幕闪白 + 持续粒子 |

---

### 紫色遗物动画序列（~2.8s）

```
T+0.0s  遮罩从 0→0.85 淡入
T+0.1s  屏幕轻微震动（±3px，0.3s）
T+0.2s  紫色光球从中心爆发，粒子向外飞散（约20个光点）
T+0.4s  遗物卡片从 scale(0.2) 弹性放大至 scale(1.05) 再弹回 scale(1.0)
        同时边缘紫光环绕（box-shadow glow 脉冲两次）
T+0.8s  卡片完全稳定，紫光余韵继续脉动
T+1.0s  遗物名称淡入上移（translateY 20px→0）
T+1.2s  稀有度标签「史诗」淡入
T+1.4s  效果描述文本逐行淡入
T+1.6s  历史典故文本逐行淡入
T+2.0s  确认按钮「收入囊中」淡入
T+2.8s  自动消失 / 点击消失
```

**紫色粒子效果 CSS 动画实现**：
- 光球：径向渐变圆 + `scale(0)→scale(1.5)` + `opacity: 1→0`
- 20个粒子点：绝对定位圆形，用 `@keyframes` 各自沿随机方向飞散，`animation-delay` 错开 0~200ms
- 震动：`@keyframes shake { 0%,100%{transform:translate(0)} 25%{translate(3px,-2px)} 50%{translate(-2px,3px)} 75%{translate(-3px,-1px)} }`

---

### 金色传说遗物动画序列（~3.5s）

```
T+0.0s  屏幕白闪（opacity 0→0.6→0，0.4s）
T+0.0s  遮罩从 0→0.95 淡入
T+0.1s  强烈震动（±6px，0.5s）
T+0.15s 底部金色光线从中心向两侧射出（8条光线，CSS conic-gradient 旋转）
T+0.3s  金色大光球爆发，粒子数量翻倍（40个），含大光斑+小火花
T+0.5s  遗物卡片从 scale(0) 弹性弹出（cubic-bezier 模拟弹性）
        卡片周围金色光晕持续脉动
T+0.9s  卡片上方浮现金色光圈/光环（装饰性）
T+1.2s  「传说」标签飞入（上移+淡入）
T+1.4s  遗物名称金光彩字淡入（text-shadow 金色辉光）
T+1.7s  套装进度条（如有）滑入
T+2.0s  效果描述、历史典故逐行淡入
T+2.5s  粒子持续漂浮（后台 loop 动画）
T+3.0s  确认按钮「收入囊中」+ 金色边框脉动
T+3.5s  自动消失 / 点击消失
```

**金色光线 CSS 效果**：
- 8条光线：`::before / ::after` + 旋转的锥形渐变，`animation: spin 2s linear infinite`
- 光球爆发：径向渐变 + `scale(0)→scale(2) opacity: 1→0`
- 金色彩字：`text-shadow: 0 0 20px #ffd700, 0 0 40px #ff8c00`

### 通用交互规则

- 动画期间背景遮罩 `pointer-events: none`，防止误触
- 点击任意位置或点击确认按钮均可关闭
- 关闭时：卡片快速缩小消失（200ms），遮罩淡出
- 紫色/金色动画关闭时额外加一次微震动反馈

## 遗物效果应用时机

| 时机 | 适用遗物 | 处理逻辑 |

| 层间转换 | 探索类、资源类 | 切换地图状态时刷新 |
